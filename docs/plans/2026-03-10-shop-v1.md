# Shop v1: Retry Shield — Implementation Plan

> **For Claude:** REQUIRED SUB-SKILL: Use superpowers:executing-plans to implement this plan task-by-task.

**Goal:** Add a "Retry Shield" consumable to the shop that lets players skip the failure animation and instantly retry an exercise.

**Architecture:** New SHOP_ITEMS entry (itemId: 2n). Shield balance in localStorage (`chesscito:shields`). PhaseFlash extended with a shield button on failure. Purchase increments balance by 3. Shield usage decrements balance and triggers immediate board reset.

**Tech Stack:** Next.js 14, Tailwind CSS, localStorage, wagmi

**Files:**
- Modify: `apps/web/src/app/play-hub/page.tsx`

**Step 1: Add shield state**

Add near the other useState declarations:

```typescript
const [shieldCount, setShieldCount] = useState(0);
```

**Step 2: Load shield count from localStorage on mount**

Add a useEffect (near the isLocalhost useEffect):

```typescript
useEffect(() => {
  try {
    const raw = localStorage.getItem("chesscito:shields");
    if (raw) setShieldCount(Number.parseInt(raw, 10) || 0);
  } catch {
    // ignore
  }
}, []);
```

**Step 3: Add helper to update shield count**

```typescript
function updateShieldCount(next: number) {
  const clamped = Math.max(0, next);
  setShieldCount(clamped);
  localStorage.setItem("chesscito:shields", String(clamped));
}
```

### Task 3: PhaseFlash Shield Button

**Files:**
- Modify: `apps/web/src/components/play-hub/mission-panel.tsx`

This is the most complex task. The PhaseFlash currently auto-fades at 700ms and hides at 1100ms. For failure with shields, we need to:
- Extend the visible duration to 3000ms (give time to tap)
- Show a shield button
- Make the overlay tappable (not pointer-events-none)

**Step 1: Update PhaseFlash props**

Change PhaseFlash from:
```typescript
function PhaseFlash({ phase }: { phase: MissionPanelProps["phase"] }) {
```

To:
```typescript
function PhaseFlash({
  phase,
  shieldCount = 0,
  onUseShield,
}: {
  phase: MissionPanelProps["phase"];
  shieldCount?: number;
  onUseShield?: () => void;
}) {
```

**Step 2: Adjust timers based on shields**

Replace the timer logic:

```typescript
useEffect(() => {
  if (!flash) {
    setVisible(false);
    setFading(false);
    return;
  }

  setVisible(true);
  setFading(false);

  const hasShield = phase === "failure" && shieldCount > 0;
  const fadeDuration = hasShield ? 2600 : 700;
  const hideDuration = hasShield ? 3000 : 1100;

  const fadeTimer = setTimeout(() => setFading(true), fadeDuration);
  const hideTimer = setTimeout(() => setVisible(false), hideDuration);

  return () => {
    clearTimeout(fadeTimer);
    clearTimeout(hideTimer);
  };
}, [phase, flash, shieldCount]);
```

**Step 3: Update the render to include shield button**

Replace the current render return (after `if (!visible || !flash) return null;`):

```tsx
const hasShield = phase === "failure" && shieldCount > 0;

return (
  <div
    className={`fixed inset-0 z-50 flex items-center justify-center bg-black/60 transition-opacity duration-400 ${
      hasShield ? "" : "pointer-events-none"
    } ${fading ? "opacity-0" : "opacity-100"}`}
  >
    <div className="flex flex-col items-center gap-4 animate-in zoom-in-90 duration-300">
      <img
        src="/art/favicon-wolf.png"
        alt=""
        aria-hidden="true"
        className="h-20 w-20 drop-shadow-[0_0_20px_rgba(103,232,249,0.5)]"
      />
      <span className={`fantasy-title text-3xl drop-shadow-lg ${flash.accent}`}>
        {flash.text}
      </span>
      {hasShield && onUseShield ? (
        <button
          type="button"
          onClick={() => {
            setVisible(false);
            onUseShield();
          }}
          className="mt-2 rounded-xl bg-amber-500/90 px-5 py-2.5 text-sm font-semibold text-white shadow-lg transition hover:bg-amber-400 active:scale-95"
        >
          Use Shield ({shieldCount} left)
        </button>
      ) : null}
    </div>
  </div>
);
```

Key changes:
- `pointer-events-none` removed when shield is available (so button is tappable)
- Shield button: amber color, shows count, calls `onUseShield` and hides the flash
- When no shield: identical to current behavior

**Step 4: Update MissionPanel to pass shield props to PhaseFlash**

Add to `MissionPanelProps`:

```typescript
shieldCount?: number;
onUseShield?: () => void;
```

Destructure them and pass to PhaseFlash:

```tsx

```
